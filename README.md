# Zeppelin Finance

## Setup & Operation

### Setup
- install node and npm 
- install hardhat globally
- create .env file (using example.env as example)

```
npm install
yarn install
```

### Compile Solidity Contracts

```
npx hardhat compile
```


### Run Unit Tests

```
npx hardhat test
```

### Run Echidna Tests
in /test directory: run echidna.sh
